STATIC INLINE uint32_t IP_REGFILE_Read ( IP_REGFILE_001_T *  pREG,
int  index 
)

Read value from General purpose registers.

Paramètres
pREG: Pointer to regfile block
index: General purpose register index
Renvoie
Read value
Note
These General purpose registers can be used to store important information when the main power supply is off. The value in these registers is not affected by chip reset. These registers are usually powered in the RTC power domain.
